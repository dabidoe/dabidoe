# RPG Character Interface

A React-based web application for interactive RPG character dialogue and abilities. Built with Vite and React.

## Features

- Interactive character card with stats (HP, AC)
- Portrait and Battle mode views
- Character dialogue system with mood indicators
- Ability buttons for character actions
- Real-time chat interface
- Beautiful gradient UI with smooth animations

## Character Abilities

- Sword Strike
- Divine Fury
- Spear Thrust
- Shield Wall
- Tell Story
- Current Quest

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Tech Stack

- React 18
- Vite 6
- CSS3 with modern gradients and animations

## Project Structure

```
src/
  components/
    CharacterCard.jsx - Main character interface component
    CharacterCard.css - Component styling
  App.jsx - Main application component
  main.jsx - Application entry point
  index.css - Global styles
```
